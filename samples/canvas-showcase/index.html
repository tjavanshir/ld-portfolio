Assessment Summary

  • Module quizzes (×5) — 25 pts (25%)
  • Discussion posts (×5) — 25 pts (25%)
  • Assignment 1: Style Analysis — 25 pts (25%)
  • Capstone Portfolio — 25 pts (25%)

Instructional Design Choices

  • Sequential gating enforces prior completion
  • Each module = learn · practise · assess cycle
  • Discussions use peer-review to scale feedback
  • H5P for interactive scenarios (no authoring tool)
  • Outcomes mapped to each module in Settings

Sample Rubric — Assignment 1: Communication Style Analysis

Criterion Excellent (5) Good (3–4) Developing (1–2)
Identification of style Correctly identifies own communication style with evidence from three or more specific examples Identifies style with one or two examples; mostly accurate Style identification is vague or unsupported
Use of theory Accurately applies at least one communication model (e.g. Shannon–Weaver); explains its relevance References a model but application is superficial No clear link to theory
Reflection quality Demonstrates critical self-awareness; identifies specific improvement strategies Some reflection; improvement strategies are generic Little evidence of reflection; surface-level response
Written communication Clear, well-structured writing; appropriate academic tone; minimal errors Mostly clear; some structural or language issues Frequent errors impede readability
Word count & format Within ±10% of 600-word target; correct APA in-text citation Slightly outside range or minor citation issues Significantly off word count or citations missing
