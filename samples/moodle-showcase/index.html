Instructional Design Note Topic-based format used to mirror job-role groupings. Completion tracking enforces sequence: each topic requires 100% completion before unlocking the next. H5P used for branching scenarios (no authoring tool licence needed). Quiz drawn from randomised question bank to reduce cheating and enable reattempts fairly. Question Bank Structure (ID Design Note) The final quiz draws randomly from a 120-question bank organised into 5 categories (1 per topic). Each attempt draws 8 questions per category = 40 questions total. This prevents rote memorisation and allows unlimited fair reattempts. All questions include specific feedback for wrong answers referencing the source page.
